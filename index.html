Houseplant Tracker App

A React project with Material UI components that keeps track of all the particulars of a user's houseplants. A registered user can add, edit, and delete plant cards on their profile. The front end uses Reqres to test registration/login and CRUD operations against an API. Test login credentials can be used to demo functionality.

API for Fitness App

RESTful API built using Express for users to keep track of their fitness classes. Full CRUD endpoints are availible to front end developers for user's fitness classes and authentication. I used JSON Web Tokens for authentication and Knex to interact with my PostgreSQL database. The API and database are hosted using Heroku CLI. Full list of endpoints and how to interact with them can be found it the GitHub readme.
